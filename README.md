# etherneter

## Instructions to setup

- Open terminal
- run `git clone https://github.com/peterlazar1993/etherneter.git`
- `cd etherneter`rm 
- Run `sudo pip3 install pipenv`
- Run `pipenv sync`


## instructions to run
- Navigate to ethernetter in CLI
- Run `pipenv shell`
- Run `sudo python <programe_name>.py`
- See output
